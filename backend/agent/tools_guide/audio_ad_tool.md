# Audio Ad Generation Guide

## Introduction
The Audio Ad Generation tool generates voiceover speech or sound clips for advertisement use, utilizing TTS (Text-to-Speech) platforms such as ElevenLabs or Google Cloud TTS, based on text scripts generated by the Text Agent.

## Steps for Generation
1. **Prepare Script**: Ensure a clear voiceover script exists (usually created by the Text Agent).
2. **Select Voice**: Match the voice persona with the target market (e.g. standard Malay voice for Malaysia, English Singlish/standard for Singapore).
3. **Local Assembly**:
   - Generate audio bytes via ElevenLabs API or Google TTS API.
   - Save the audio bytes to a local temporary file (.mp3).
4. **S3 Upload**: Upload the audio file to AWS S3 under `generated_ads/{project_id}/{task_id}/{filename}.mp3`.
5. **Supabase Record**: Insert a new record into `public.generated_ads` with type `audio` and update the canvas node state.
