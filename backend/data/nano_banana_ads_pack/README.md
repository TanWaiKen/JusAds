# Nano Banana / Veo / Seedance / GPT Image Ads Pack

This pack contains:
- 10 Veo-ready video ad prompts
- 10 Seedance-ready video ad prompts
- 10 GPT Image / Nano Banana-ready image ad prompts

## Prompt rules used
- Veo prompts are detailed, specifying framing, camera movement, lighting, scene details, action, and optional audio/dialogue.
- Seedance prompts follow subject + motion + scene + camera language, and avoid relying on negative prompts because the guide says negative prompts do not take effect.
- Image prompts are written in a structured commercial style for ad generation, with clear subject, setting, lighting, composition, and copy-safe empty space.

## Folder format
Each ad folder includes:
- ad-brief.md
- prompt.txt
- notes.md
- output/.gitkeep
