# Veo prompt template

Format your prompt like this:

Type of video + subject + action + scene details + framing + camera movement + lighting + style + ending frame + optional audio/dialogue.

Example skeleton:
Create a [duration] [aspect ratio] [style] ad video for [PRODUCT]. Show [SUBJECT] doing [ACTION] in [SETTING]. Use [FRAMING] and [CAMERA MOVEMENT]. Lighting is [LIGHTING]. The mood is [MOOD]. End on [FINAL FRAME]. Audio: [AUDIO].
